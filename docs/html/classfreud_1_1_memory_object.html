Detailed Description

template<typename T>
class freud::MemoryObject< T >

The base class for all MemoryObjects.

All MemoryObjects used in freud should derived from this type. For example, if a user wanted to locate a structure with the following layout:

struct Position {
int x;
int y;
}

If the user knew that 'x' would always equal '10', they could create a MemoryObject with this constraint as follows:

class PositionMatcher : public MemoryObject<Position> {
static bool verify(const Position& p) {
return p.x == 10;
}
}
