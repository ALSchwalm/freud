Detailed Description

template<typename MemObject>
class freud::MemoryContextIterator< MemObject >

An iterator into some MemoryContext.

MemoryContextIterators apply a typed view onto a MemoryContext. When dereferenced, the iterator returns a reference to an instance of the MemObject::type for which MemObject::verify returns true.
